MINIKUBE commands:
minikube
    version
    start --memory=4g !!! // kubectl will not work without it
    stop
    status
    ip - master node ip address
    service --url {serviceName}
    tunnel // to get access service on specified port (i.e. zipkin)

KUBECTL commands:
kubectl
    get pods
    get services
    port-forward pod/hello-world 8080:80
    delete pod {podName}
    describe pod {podName}
    logs {podName}
    get all
